# 微写作系统

## 简介

本系统提供Docker模式部署，当前运行端口为`36790`，可在创建时进行修改。

## 部署

```sh
cd microwrite
docker build  -t thucc-microwrite:v0.0.1 .
docker run -d -p 36790:80 --name thucc-microwrite thucc-microwrite:v0.0.1
```

## 调用

```python
import json
import requests

def microwrite(prompts):
    """
```
